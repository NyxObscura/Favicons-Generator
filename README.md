# Favicon Generator

**Favicon Generator** is a powerful Node.js script built on Sharp and png-to-ico that automatically generates a variety of favicons from a single source image. Perfect for web developers looking to streamline their workflow and ensure their site has all the necessary icons in one go!

---

## ✨ Features

✅ **Generate multiple favicons**, including:

- `favicon.ico` (16x16 & 32x32)
- `favicon-16x16.png`
- `favicon-32x32.png`
- `android-chrome-192x192.png`
- `android-chrome-512x512.png`
- `apple-touch-icon.png`
- `site.webmanifest`

✅ **Automatically detects input image** from the root folder (`photo.png` or `photo.jpg`).  
✅ **Saves generated favicons** to the `favicons/` folder.  
✅ **Easy to use** – just one command to get started!  

---

## 📥 Installation

1️⃣ **Clone or download this repository:**

```bash
git clone https://github.com/NyxObscura/Favicons-Generator.git
cd Favicon-Generator
```

2️⃣ **Install dependencies:**

```bash
npm install
```

---

## � Usage

1️⃣ Place your favicon image in the root folder with the name `photo.png` or `photo.jpg`.  
2️⃣ Run the following command:

```bash
npm start
```

3️⃣ Your favicons will be generated and saved in the `favicons/` folder.  

---

## 🛠 Dependencies

- **Sharp** – For resizing images.  
- **png-to-ico** – For creating `.ico` favicons.  
- **fs-extra** – For managing files and directories.
